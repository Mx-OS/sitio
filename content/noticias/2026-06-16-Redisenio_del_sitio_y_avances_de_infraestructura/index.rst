¡Hola a todos!

Hoy nos complace compartir un par de anuncios importantes sobre el progreso del proyecto MxOS:

Rediseño del sitio web
======================
Hemos renovado completamente nuestro sitio web oficial para ofrecer una experiencia más moderna, limpia y minimalista, utilizando
`Pico CSS <https://picocss.com/>`_. Este nuevo tema es sumamente ligero, optimizado para dispositivos móviles y estructurado
pensando en un diseño semántico y eficiente.

Cabe destacar que el sitio sigue estando construido con el generador de sitios estáticos `Hugo <https://gohugo.io/>`_. Aprovechamos
esta oportunidad para mencionar que **seguimos en búsqueda activa de un webmaster** que quiera sumarse al proyecto para ayudarnos a
mantener y mejorar continuamente nuestra presencia web.

Actualizaciones de Infraestructura
==================================
Adicionalmente, hemos reanudado formalmente el desarrollo y las pruebas en dos de nuestros componentes clave de infraestructura:

* **rpms-sync**: Retomamos el trabajo en nuestra herramienta para sincronizar paquetes RPM de manera automatizada. Puedes ver el
  repositorio en: https://git.mx-os.mx/MxOS/rpms-sync
* **ansible-koji**: Continuamos afinando nuestro playbook de despliegue para el sistema de construcción de paquetes Koji. Puedes
  ver el repositorio en: https://git.mx-os.mx/MxOS/ansible-koji

Agradecemos enormemente su apoyo continuo. Si deseas unirte a las discusiones o colaborar directamente con el equipo de
infraestructura, recuerda que nos encontramos en nuestro canal de `Telegram <https://t.me/mx_os_mx>`_.

¡Sigamos construyendo la soberanía tecnológica!
